Full credits to the original contributors for an amazing template. 

## Steps to Run

1. Download Docker and install the devcontainers plugin in VSCode
2. Use `Ctrl/Cmd+Shift+P` to build and reopen inside the container in VSCode
3. Run `bundle exec jekyll serve`
